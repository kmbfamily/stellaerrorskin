xe stella 오류 스킨
===============
## 기능 ##
 * xe 홈페이지에서 404 오류 발생시 밋밋한 기본스킨을 대신하여 화려하고 감성적인 페이지 제공
## 특징 ##
 * pc버전과 모바일 버전과 동일한 404 페이지를 제공합니다.(모바일용 스킨도 설치시)
 * 모바일용 스킨은 소셜로그인 모듈과의 호환성 확보
## 설치경로 ##
- modules/message/skins
## 설치방법 ##
- ftp로 xe core가 설치된곳에 있는 설치경로 안에 stella 폴더를 통째로 업로드 합니다.
